Process

How SOUNDPATH works

Four intelligent steps from profile to personalised career intelligence.

1

Build Your Profile

Enter your genre, career stage, goals, and upload your metrics. Our Node.js backend securely processes and normalises your data.

2

ML Processing

Python ML models (scikit-learn + TensorFlow) score your career trajectory, identify gaps, and cluster you with comparable artists.

3

LLM Synthesis

A fine-tuned large language model transforms raw model outputs into human-readable insights, advice, and personalised strategy.

4

Act on Insights

Access your roadmap, chat with AI advisor, track progress, and watch recommendations update in real time as your career evolves.

Technology Stack

Built with cutting-edge AI

A full-stack AI system combining web technologies, machine learning, and large language models.

HTML5 + CSS3

Semantic, accessible front-end markup with custom design system, CSS variables, and responsive layouts optimised for all devices.

Node.js Backend

Express.js REST API handles authentication, data routing, LLM proxy, WebSocket connections for real-time updates, and session management.

Python ML Engine

FastAPI service running scikit-learn, TensorFlow, and Pandas pipelines. Handles career scoring, clustering, recommendation, and forecasting models.

Large Language Model

LLM integration (Claude / GPT-4) with custom system prompts, RAG over music industry knowledge base, and structured output parsing.

Machine Learning

Random forest, gradient boosting, and neural network models trained on anonymised career outcome data from 50,000+ music professionals.

Real-Time Data

Streaming API integrations, web scraping via Python (BeautifulSoup + Playwright), and a PostgreSQL database with vector embeddings (pgvector).
